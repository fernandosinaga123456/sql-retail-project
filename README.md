# sql-retail-project
Data penjualan retail bukan sekadar angka. Di baliknya ada pola yang bisa jadi keputusan bisnis. Proyek ini menggali data tersebut menggunakan MySQL untuk menjawab pertanyaan bisnis nyata yang relevan bagi operasional dan strategi perusahaan, melalui query SQL mencakup joins, aggregations dan subqueries.
